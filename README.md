# 창문형 에어컨 설치 안내 웹앱

> 고객이 URL/QR 코드로 접속하면 별도 앱 설치 없이 브라우저에서 바로 사용하는 창문형 에어컨 설치 안내 웹앱

---

## 🌐 접속 URL

| 페이지 | 경로 | 설명 |
|--------|------|------|
| 설치 안내 메인 | `/index.html` | 단계별 설치 안내 시작 |
| QR 코드 페이지 | `/qr.html` | QR 코드 확인 및 인쇄 |

---

## ✅ 구현 완료 기능

### 단계별 안내 흐름 (0~6단계)

| 단계 | 화면 | 설명 |
|------|------|------|
| 0 | 시작 화면 | 앱 소개 및 시작 버튼 |
| 1 | 창문 유형 선택 | 미닫이 / 여닫이 선택 |
| 2 | 창문 크기 입력 | 가로·세로(cm) 입력 및 실시간 검증 |
| 3 | 제품 유형 선택 | 일반(92cm 이상) / 미니(77cm 이상) + 높이 범위 선택 |
| 4 | 창틀 깊이 선택 | 2cm 미만/이상 선택, b·c 치수 추가 입력 |
| 5 | 특수 창문 유형 | 나무 창틀 / 분할 창문 / 두꺼운 창틀 다중 선택 |
| 6 | 결과 화면 | 설치 가능 여부, 필요 키트, 호환 모델, 특이사항, 설치 방법 |

### 자동 계산 로직

- **열리는 창문 가로**: 38cm 이상 조건 체크
- **창문 높이**: 일반 제품(92cm 이상) / 미니 제품(77cm 이상) 조건 체크
- **창틀 깊이**: 2cm 이상/미만 분기
- **b+c 치수**: b(1cm 이상 2cm 미만) + c(1.2cm 이상) 조건 체크
- **추가 키트 필요 여부**: 높이 범위에 따라 자동 판단
- **두 번째 설치 브라켓**: 창틀 깊이 2cm 미만 시 자동 표시

### 제품 데이터 (더미)

| 키트 종류 | 호환 모델 |
|-----------|-----------|
| 듀얼인버터 추가 키트 | PWA-2200, PWA-2250, PWA-3200, PWA-3250, PWA-3300, PWA-M3100 |
| 프리미엄 추가 키트 | PWA-3400, PWA-3500, PWA-3600, PWA-M3500 |

### UI/UX 기능

- 모바일 우선(max-width: 480px) 반응형 레이아웃
- 진행 바 (5단계 기준 퍼센트 표시)
- 이전/다음/완료 버튼 내비게이션
- 실시간 입력값 검증 (success/error 상태 표시)
- 토스트 알림 메시지
- Web Share API 지원 (결과 공유)
- QR 코드 생성 및 인쇄 기능
- iOS Safe Area 대응

## 📱 QR 코드로 배포하기

1. **Publish 탭**에서 프로젝트를 배포하면 URL이 생성됩니다.
2. `/qr.html`에 접속하면 해당 URL로 연결되는 QR 코드가 자동 생성됩니다.
3. QR 코드를 인쇄하거나 이미지로 저장하여 고객에게 제공하세요.
4. 고객은 스마트폰으로 QR 코드를 스캔하면 앱 설치 없이 바로 이용 가능합니다.
