# 창문형 에어컨 설치 안내 웹앱

> 고객이 URL/QR 코드로 접속하면 별도 앱 설치 없이 브라우저에서 바로 사용하는 창문형 에어컨 설치 안내 웹앱

---

## 🌐 접속 URL

| 페이지 | 경로 | 설명 |
|--------|------|------|
| 설치 안내 메인 | `/index.html` | 단계별 설치 안내 시작 |
| QR 코드 페이지 | `/qr.html` | QR 코드 확인 및 인쇄 |

---

## ✅ 구현 완료 기능

### 전체 화면 흐름

| 화면 | 내용 |
|------|------|
| 시작 화면 | 앱 소개 및 시작 버튼 |
| 모델 선택 | 정속형/인버터·프리미엄·듀얼인버터·하이브리드 4가지 선택 |
| 1단계 | 창문 유형 선택 (미닫이 → 정상 진행 / **여닫이 → 설치 불가 안내 화면 분기**) |
| 2단계 | 가로·세로(cm) 입력 + 실시간 조건 검증 |
| 3단계 | 일반/미니 제품 선택 + 높이 범위 → 키트 필요 여부 안내 |
| 4단계 | 창틀 깊이 2cm 미만/이상 선택 → b+c 치수 추가 입력 |
| 5단계 | 창문 환경 **4가지 카드** 선택 (플라스틱 샷시/두꺼운 창틀/나무 창틀/분할 창문) |
| 결과 | 설치 가능 여부, 필요 키트, 호환 모델, 특이사항, 설치 방법, **유튜브 설치 가이드 영상** |
| 여닫이 불가 | 파스텔 배경 안내 + 처음으로 돌아가기 버튼 |

### 유튜브 설치 가이드 매핑

| 모델 그룹 | 창문 환경 | 제공 버튼 |
|-----------|-----------|-----------|
| 정속형/인버터 (A), 듀얼인버터 (C) | 플라스틱/두꺼운/나무/분할 | 기본 설치 · 두꺼운 창틀 · 긴 창문 · 나무 창틀 · 통합 가이드 |
| 프리미엄 (B), 하이브리드 (D) | 플라스틱/두꺼운/나무/분할 | 프리미엄 기본 · 긴 창문 · 나무 창틀 · 통합 가이드 |

### 제품 데이터

| 그룹 | 라벨 | 모델 |
|------|------|------|
| A | 정속형/인버터 | PWA-2100, PWA-2200, PWA-2250, PWA-3200, PWA-3250 |
| B | 프리미엄 | PWA-3400, PWA-3500, PWA-3600, PWA-M3500, PWA-3301BE2 |
| C | 듀얼인버터 | PWA-3100, PWA-3300, PWA-3300BE |

```

├── manifest.json         # PWA 매니페스트

│   └── style.css         # 모바일 우선 스타일

├── images/
│   ├── window-measure.png  # 창문 측정 방법 (38cm 기준)
│   ├── sill-depth.png      # 창틀 단면 도면 (a/b/c 치수)
│   └── split-window-kit.png # 분할 창문 설치 비교 (X/O)

```

## 🚀 배포 및 QR 코드 제공 방법

1. **Publish 탭**에서 배포 → 라이브 URL 생성
2. `/qr.html` 접속 → 라이브 URL이 QR 코드로 자동 생성
3. QR 코드를 인쇄하거나 이미지 저장 후 고객에게 제공
4. 고객은 스마트폰 카메라로 스캔 → 앱 설치 없이 바로 이용
